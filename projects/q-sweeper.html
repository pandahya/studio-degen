Q-Sweeper

2021

Q-sweeper stands for quantum minesweeper. The traditional game is redesigned by applying quantum concepts to it, such as noise and entanglement. The maps are generated using data from a quantum algorithm to place the mines in the game.
For next steps, we are exploring entanglement further by using socket connections to entangle multiple ongoing games. This project is being set up for an interactive installation that further explores the intersections of games and quantum computing.
The code is open-source and the game can be played in the browser.
